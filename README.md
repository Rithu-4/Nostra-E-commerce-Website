# Nostra – E-commerce Website

Nostra is a responsive e-commerce website developed using HTML, CSS, and JavaScript. The website provides a simple and modern online shopping experience where users can explore products, view collections, and contact the store easily.

**Pages Included**
Home Page

The home page introduces the website with a clean and attractive layout, showcasing featured products and sections that help users easily explore the store.

Collection Page

The collection page displays different products in an organized grid layout, allowing users to browse and explore available items conveniently.

Contact Page

The contact page provides a simple contact form and basic information for users to get in touch with the store.

**Features
**
Fully responsive design that works smoothly on mobile, tablet, and desktop devices

Clean and modern user interface

Easy navigation between pages

Interactive elements implemented using JavaScript

**Technologies Used**

HTML – Website structure

CSS – Styling, layout, and responsive design

JavaScript – Interactivity and dynamic features
**
Purpose**

This project was created to practice front-end web development skills and to understand how a responsive multi-page e-commerce website works.
